C Programming  
Repository by ak26009  

Purpose  
This repository contains C language programs focused on foundational concepts and problem-solving.
C skills: data types, control flow, functions, pointers, arrays, file I/O, etc.

Repository Structure  
- `Basics/` — Simple programs: print statements, data types, operators.  
- `Control‐Flow/` — Code for `if/else`, `switch`, loops (`for`, `while`, `do‐while`).  
- `Functions_Pointers/` — Examples of defining/using functions, passing by reference, pointers.  
- `Arrays_Strings/` — Array operations, string manipulation, sorting/searching.  
- `File_IO/` — Programs that read/write files, process file contents.  
- `Advanced/` — More complex topics: dynamic memory, structures, unions, linked lists.
